## Instalación
```
 git clone http://github.com/micuenta/mirepo.git
 cd mirepo
 npm install
 npm start
```
## Tecnologías utilizadas
- react 17+
- node-sass 7+
- react-dom 17 +
- react-router-dom 6+
- react-scripts 5+
- firebase 9+
- react-toastify 8+
